# CybersecurityAwarenessBot

## Project Overview
The Cybersecurity Awareness Bot is an interactive tool designed to educate users about digital safety.It is a .NET based bot designed to help user stay safe online by learning about cybersecurity best practices through interactive messages with the bot.

## Features
- Voice greeting using a WAV audio file
- An ASCII art logo as a introduction banner
- User name prompt and personalized user name greeting
- Instruction to assit user to navigate through
- Chatbot responses to user questions
- Console user interface with different colour and dividers
- Input validation for empty or incorrect input

## Cybersecurity Topics 
- Phishing attacks
- Password safety
- social engineering
- safe browsing
- general cybersecurity tips
- cyber attacks

## Technologies used
- .NET framework
- visual studio C#
- GitHub
- GitHub action for continious intergration
